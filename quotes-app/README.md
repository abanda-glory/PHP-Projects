# Quotes App

A simple Laravel web application that displays random motivational quotes and allows users to submit their own.

## Project Overview

Quotes App is a beginner-friendly Laravel project built to practice core Laravel concepts such as:

- Routes
- Controllers
- Blade templates
- Forms
- Database (MySQL)
- Migrations

This project is designed as a simple 1-page motivational quotes app.

## Features

- Display one random motivational quote on page load
- Show a new random quote with a **New Quote** button
- Allow users to submit their own quote
- Store quotes in a MySQL database

## Tech Stack

- Laravel
- PHP 8.5
- MySQL
- Blade
- XAMPP
- Git & GitHub
